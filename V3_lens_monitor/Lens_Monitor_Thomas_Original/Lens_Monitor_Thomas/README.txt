Lens monitor firmware for use on a merlin lens controller board.

Test lens voltages need to be loaded into the test voltage test header file in /Src.
These lens voltages will be applied to the corresponding lens at time intervals specified inside the header file.

STM studio can be used to monitor the lens voltages and currents including logging and calculation stats.
